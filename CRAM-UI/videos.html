Videos of the Course Resource Appraisal Model

Overview
This video provides an overview of the major features of the CRAM tool. It illustrates how the view the tool can provide after the data has been elicited from the user.

Basic Module Data
In this video we concentrate on the data that describes the module, such as student cohort numbers, income and costs.

Modelling Learning Experiences
This video demonstrates how we can visualise and interact with the characteristics of the module that describe the learning experiences that students expect from a module.

Modelling Teaching Costs
In this final video we focus on the way in which the CRAM tool can provide an insight into the costs of preparing and supporting a module.
